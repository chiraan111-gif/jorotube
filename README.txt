JoroTube Plus (FB + YT + WA features)

Run locally:
  python -m venv .venv
  .venv\Scripts\activate (Windows)  OR  source .venv/bin/activate (Linux/Mac)
  pip install -r requirements.txt
  set FLASK_ENV=production
  python run.py

Login/Signup at /register and /login
Routes: / (home), /watch/<id>, /upload, /reels, /friends, /messages, /notifications
